# FIEA Game Engine & Binding Of Isaac Remake (Private Repository)

## **<u>Full private project available to employers upon request (private for academic integrity).</u>**

Over a 3-month period, I created a custom game engine in C++. This included a custom service manager alongside a memory service, layer-based collision service, and event service with carefully designed abstracted interfaces. The core feature was a runtime reflection system used to enable a custom JSON-driven scripting schema for authoring game objects and behaviors. These features were tested rigorously through 25,000+ lines of unit test and integration test code.

Finally, with a team of 5 programmers, we used my custom game engine to re-create Binding of Isaac, a top-down roguelike shooter.
